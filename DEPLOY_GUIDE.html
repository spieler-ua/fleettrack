Розмістіть файли онлайн
GitHub Pages — безкоштовно, назавжди, без сервера
💡
Telegram Mini App вимагає HTTPS-посилання на ваш застосунок. GitHub Pages надає його безкоштовно.
📁 Файли які потрібно завантажити
  • 📄
    index.html — головний застосунок
  • 📄
    manifest.json — PWA маніфест
  • 📄
    sw.js — Service Worker
  • 📁
    icons/ — папка з іконками (icon-192.png, icon-512.png)
🐙 Кроки на GitHub Безкоштовно 5 хвилин
  • 1
    Відкрийте 🔗 github.com і зареєструйтесь (або увійдіть)
  • 2
    Натисніть New repository (зелена кнопка вгорі справа)
    Назва: fleettrack · Visibility: Public · Create repository
  • 3
    На сторінці репозиторію натисніть uploading an existing file
    Перетягніть всі 4 файли + папку icons/ → Commit changes
  • 4
    Перейдіть до Settings (вкладка вгорі) → розділ Pages (ліве меню)
    Source: Deploy from a branch · Branch: main · Folder: / (root) → Save
  • 5
    Зачекайте 1-2 хвилини. З'явиться ваш URL:
    https://ВАШ-ЛОГІН.github.io/fleettrack/
    Збережіть цей URL — він знадобиться в кроці 3
⚠️
Якщо папку icons/ не вдається завантажити через браузер — скористайтесь GitHub Desktop (безкоштовна програма) або просто перетягніть файли icons прямо в кореневу папку репозиторію.

Створіть бота в Telegram
Через офіційного @BotFather — 2 хвилини
🤖 Розмова з @BotFather
  • 1
    Відкрийте Telegram, знайдіть @BotFather (синя галочка ✓)
    Натисніть START
  • 2
    Надішліть команду: /newbot
  • 3
    Назва бота (відображається в чаті):
    Наприклад: FleetTrack GPS
  • 4
    Username бота (має закінчуватись на _bot):
    Наприклад: fleettrack_gps_bot
  • 5
    BotFather дасть вам TOKEN — довгий рядок вигляду:
    7123456789:AAFxxx...yyy
    ⚠️ Збережіть токен у безпечному місці, нікому не передавайте

Підключіть Mini App до бота
Налаштування кнопки меню і Web App
💡
Є два способи — Menu Button (кнопка біля поля вводу) і /newapp (окремий застосунок). Рекомендуємо обидва.
🔘 Спосіб А — Кнопка меню (найшвидше)
  • 1
    В чаті з @BotFather надішліть: /mybots
  • 2
    Виберіть вашого бота → Bot Settings → Menu Button
  • 3
    Configure menu button
    URL: https://ВАШ-ЛОГІН.github.io/fleettrack/
    Text: 🚗 Відкрити FleetTrack
✅
Тепер у чаті з ботом з'явиться кнопка 🚗 Відкрити FleetTrack поруч з полем вводу.
📱 Спосіб Б — Окремий Mini App (опціонально)
  • 1
    В @BotFather надішліть: /newapp
  • 2
    Виберіть вашого бота зі списку
  • 3
    Title: FleetTrack
  • 4
    Description: GPS відстеження маршрутів водіїв
  • 5
    Web App URL: https://ВАШ-ЛОГІН.github.io/fleettrack/
  • 6
    Завантажте іконку (будь-яке квадратне фото 640×640px)
    Ви отримаєте посилання: https://t.me/ваш_бот/fleettrack
💬 Додайте команду /start з кнопкою (опціонально)

Щоб при /start одразу відкривався застосунок — використайте inline keyboard через Telegram API:

# Викликати через браузер (замінити TOKEN і URL): https://api.telegram.org/botTOKEN/setMyCommands ?commands=[{"command":"start","description":"Відкрити FleetTrack"}] # Відправити повідомлення з кнопкою (через curl або Postman): POST https://api.telegram.org/botTOKEN/sendMessage { "chat_id": ВАШ_CHAT_ID, "text": "🚗 FleetTrack готовий до роботи!", "reply_markup": { "inline_keyboard": [[{ "text": "Відкрити застосунок", "web_app": { "url": "https://ВАШ-ЛОГІН.github.io/fleettrack/" } }]] } }

Перевірка і поширення
Фінальні кроки

  • ☐

  • ☐

  • ☐

🔗 Як поділитись з водіями
  • 1
    Надішліть водіям посилання на бота: https://t.me/ваш_бот
  • 2
    Або якщо створили Mini App: https://t.me/ваш_бот/fleettrack
  • 3
    Водій натискає START → кнопка меню → вводить ім'я → починає поїздку
📌
Ім'я водія підтягується автоматично з профілю Telegram — водію не треба нічого вводити вручну.
🛠 Поширені проблеми
  • ❓
    Застосунок не відкривається в Telegram
    Перевірте що URL починається з https:// (не http)
  • ❓
    GPS не працює в Mini App
    На iOS: Налаштування → Telegram → Геолокація → «При використанні»
    На Android: Telegram → Дозволи → Місцеположення → Дозволити
  • ❓
    GitHub Pages показує 404
    Зачекайте 2-3 хвилини після збереження налаштувань Pages. Іноді потрібен перший commit після увімкнення.
  • ❓
    Карта не завантажується
    Перевірте підключення до інтернету. OpenStreetMap потребує мережі при першому відкритті.
